Цей фан проєкт — концепція "AI-Оперованої Системи" (AI OS), де людина виступає архітектором, а ШІ — виконробом, який пише, тестує та розгортає код у реальному часі. Досить класний досвід можна отримати, якщо спробувати написати це самостійно, або навіть залайвкодити.
Суть проекту полягала в тому, щоб дізнатись як складно створити проект на базі ШІ, який буде не тільки видавати тестову відповідь, а, фактично, робити всі етапи структуризації та тесту за людину
В моєму випадку вийшов базовий етап коду, який бере API key і за допомогою нього висиляє промпт до ШІ, в цей же час генерується запит у вигляді JSON і через функцію execute_plan_list(plan) перетворює запит на реальні дії
Деякі результати були завантажені в гіт (Tic-Tac-Toe або SystemMonitor)
Під модифікації код реальний, головне вільний час і API з незкінченними токенами :)
